# 01 — Gestión de archivos

Navegación del sistema de archivos, manipulación de archivos y búsqueda con find.

## Comandos cubiertos

- `pwd`, `ls`, `cd` — navegación básica
- `touch`, `echo`, `cat` — crear y leer archivos
- `cp`, `mv`, `rm` — copiar, mover y eliminar
- `find` — búsqueda avanzada con criterios múltiples
