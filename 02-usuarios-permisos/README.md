# 02 — Usuarios y permisos

Permisos de archivos, ownership y configuracion de umask en Rocky Linux.

## Comandos cubiertos

- `ls -la` — leer permisos en detalle
- `chmod` — modificar permisos en forma simbolica y numerica
- `chown` — cambiar dueño y grupo de archivos
- `umask` — ver y modificar permisos por defecto
- `id` — ver usuario actual y grupos a los que pertenece
- `groups` — ver grupos del usuario actual
